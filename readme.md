## 📝 Deskripsi
Ini adalah edisi pertama Kamus Pemasyarakatan bahasa Indonesia - Inggris yang dibuat khusus untuk membantu penerjemahan istilah teknis di lingkungan pemasyarakatan.

## 🚀 Fitur Utama
- **Pencarian Cepat:** Mencari istilah dalam Bahasa Indonesia maupun Inggris.
- **Definisi Lengkap:** Menyediakan konteks hukum untuk setiap istilah.
- **Update Mandiri:** Fitur "Add New Term" memungkinkan penambahan kosakata baru langsung dari aplikasi.
